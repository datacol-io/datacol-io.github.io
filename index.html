Why Razorbox

Private and Secure

Razorbox provision isolated and single tenant hosted environment in your GCP account with recommended configuration including Kubernetes, Private network. Your data and code will always be inside your account.

Productive and Powerful

Become productive and accelerate your workflow using developer tools and service integrations.

Optimized for GCP

Razorbox optimize your environment with best of community knowledge providing tight integration with cloud services and sane defaults

How it works

1

Setup

Get start by downloading rz cli to create a cluster

will spin up a kubernetes cluster and other resources in your GCP account. You can deploy multiple apps in a cluster and share services among them.

2

Build & Deploy

Create 12-factor apps with Buildpacks, Dockerfile or docker-compose.yml on Kubernetes

Provision infrastu with helm charts managed by razorbox

Link GCP services easily (like automatic cloud-sql proxy setup, service-account linking)

3

Manage & Scale

Automatically setup logging, metrics and alerts for your applications to become more productive Configure Auto scaling Execute scheduled and one-off tasks HTTPS/SSL certificates

Features

Container native

Use Docker images, containers and buildpacks out of the box

Optimized for GCP

We integrate cloud services with recommended infra settings.

CLI and CI/CD tool

Use developer friendly cli tool to configure and manage your apps

Productivity tools

Integrate notifications with productivity apps like slack.

Open source

Razorbox source code is open-sourced at github and open to suggesstions and contributions.

Portable

While Razorbox is tied with Google, your apps are still portable, and run elsewhere unlike App-engine.
