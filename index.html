How It Works

  • Install
  • Build & Deploy
  • Manage & Scale

{{site.title}} is a control layer on top of cloud(AWS/GCP), to provide Heroku like experience with power of Kubernetes under the hood.

Get started by downloading our CLI to create a stack. It will create a Kubernetes cluster and other resources in your account.

You can deploy multiple apps in cluster and share services among them.

{% highlight shell %} curl -sSL https://www.datacol.io/install | sh {% endhighlight %} {% highlight shell %} datacol init {% endhighlight %}

Build your apps and easily deploy them to cluster. You can -

  • Create 12-factor apps with Buildpacks, Dockerfile or docker-compose.yml on Kubernetes.
  • Provision infrastructure with helm charts managed by us.
  • Link GCP services easily (like automatically setting cloud-sql proxy , service-account and many more).
{% highlight shell %} datacol apps create # create app {% endhighlight %} {% highlight shell %} datacol deploy # deploy app {% endhighlight %} {% highlight shell %} datacol env set API_KEY=secret # set env vars {% endhighlight %}

We will automatically setup logging, metrics and alerts to become more productive.

  • Configuring Auto scaling
  • Executing scheduled and one-off tasks
  • HTTPS/SSL certificates using ACME
{% highlight shell %} # install redis as backing service datacol helm install stable/redis {% endhighlight %} {% highlight shell %} datacol logs -t # stream logs {% endhighlight %} {% highlight shell %} datacol scale --count=3 # scale {% endhighlight %}

Get Started

Getting started with {{site.title}} is easy. It will take almost 10 minutes to go from zero to first successful deployment.

1. Download CLI

{% highlight shell %} > curl -sSL https://www.datacol.io/install | sh {% endhighlight %}

2. Create a new stack

{% highlight shell %} > datacol init --stack demo {% endhighlight %}

It will create a new stack by the name demo .

3. Deploy a sample app

{% highlight shell %} > git clone https://github.com/datacol-io/samples.git && cd samples/node-demo > STACK=demo datacol apps create > datacol deploy > curl http://:8080 hello World {% endhighlight %}
