Try {{site.title}}

Get started with {{site.title}} is easy. It will take almost 10 minute to go from zero to first deployment.

1. Download cli

{% highlight shell %} curl -Ls https://storage.googleapis.com/datacol-distros/osx > /usr/local/bin/datacol {% endhighlight %}

2. Create a new stack

Get project identifier from Google Console or create a new one.

{% highlight shell %} datacol init --project gcp-demo --stack demo --zone us-east1-b {% endhighlight %}

It will create a new stack by name "demo" in us-east1-b zone under gcp-demo project.

3. Deploy your app

{% highlight shell %} git clone git@github.com:datacol-io/node-demo.git && cd - datacol apps create datacol deploy {% endhighlight %}

If you face any difficulty, please don't hesitate to reach out or send your suggestions here .
